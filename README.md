# Complaint-Management-System
Agile Development Project using .NET Core, ASP.NET Web API, Angular/React, SQL Server, Docker, Kubernetes, and Azure(option scalable).

## Tech Stack
- .NET Core
- ASP.NET Web API
- Angular / React
- SQL Server
- Docker
- Kubernetes (AKS)
- Microsoft Azure
- CI/CD Pipeline

## Architecture
Client → Angular/React → ASP.NET Web API → SQL Server → Docker → Kubernetes → Azure
